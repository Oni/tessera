tessera by jdp

Tessera is an extremely tiny PHP framework modeled after web.py, Sinatra, and Juno. It allows the developer to map URIs to class methods while keeping their code DRY, and separating application logic and presentation. Its feature set is small and strong: routes with named params (and regular expressions and splats!) with view and layout logic.

An Example Application

A Tessera application is just a single PHP class, with an array of routes that define which methods handle requests. The most simple application looks like this: 

<?php
require 'tessera.php';

class BasicApp extends Tessera {
    function index() {
        echo "Hey little world!";
    }
}

$basic = new BasicApp(array(
    '/' => 'index'
));
